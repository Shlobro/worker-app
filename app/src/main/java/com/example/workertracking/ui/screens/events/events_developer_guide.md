# Events Screen Developer Guide

This package handles screens related to Event management.

## Files

- **EventsScreen.kt**: List of all events.
- **EventDetailScreen.kt**: Details view for a single event.
- **AddEventScreen.kt**: Form to create a new event.
- **EditEventScreen.kt**: Form to edit an existing event.
- **AddWorkerToEventScreen.kt**: Screen to assign workers to a specific event.
