# Components Package Developer Guide

This package contains reusable Jetpack Compose UI components.

## Files

- **AddWorkerDialog.kt**: A dialog for adding a new worker.
- **Cards.kt**: Contains card layouts used for displaying items in lists (e.g., WorkerCard, EventCard).
- **EditEventWorkerDialog.kt**: Dialog for editing a worker's assignment to an event.
- **PaymentDialogs.kt**: Dialogs related to recording payments.
- **SearchableEmployerSelector.kt**: A dropdown/search component for selecting an employer.
- **SearchableWorkerSelector.kt**: A dropdown/search component for selecting a worker.
- **SharedComponents.kt**: General purpose small components (e.g., standard buttons, text fields) reused across the app.
